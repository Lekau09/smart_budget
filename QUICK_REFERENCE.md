## ðŸŽ¯ Common Tasks (How to Do Them)

### Task: Set Your Monthly Budget
**Location:** Dashboard
**Steps:**
1. Click "Set Budget" button (top right)
2. Enter your monthly limit (e.g., M10,000)
3. System calculates automatically:
   - Spent = All your expenses
   - Remaining = Budget - Spent
   - Saved = Allocated to goals

### Task: Add a New Expense
**Location:** Dashboard or Transactions page
**Steps:**
1. Click "Add Expense" button
2. Fill in:
   - Amount (M)
   - Category (dropdown)
   - Description (optional)
   - Date (auto-filled with today)
3. Click "Add"
4. All pages update automatically

### Task: Create a Savings Goal
**Location:** Savings page
**Steps:**
1. Click "Add New Goal" section
2. Fill in:
   - Goal Name (e.g., "Vacation")
   - Target Amount (e.g., M5,000)
   - Current Amount (optional)
3. Click "Create Goal"
4. Goal appears in grid with 0% progress

### Task: Add Money to a Savings Goal âœ¨
**Location:** Dashboard or Savings page
**Steps:**
1. Find your goal card
2. Click "+ Add Savings" button
3. Modal appears showing:
   - Current saved amount
   - Target amount
   - Progress %
4. Enter amount to add (e.g., M500)
5. Click "Add Savings"
6. Progress bar updates immediately
7. See success message

### Task: View Your Spending Breakdown
**Location:** Dashboard or Analytics
**Steps:**
1. **Dashboard:** See pie chart instantly
2. **Analytics:** Detailed category table + trends
3. Colors show which categories use most budget
4. Can see percentages and amounts

### Task: Track Multiple Goals
**Location:** Savings page
**Steps:**
1. View all goals in 3-column grid
2. Each shows: Name, %, saved/target, remaining
3. Color coding: Blue (0-50%) â†’ Amber (50-75%) â†’ Green (75%+)
4. Click "+ Add Savings" on any goal
5. Repeat as needed for multiple goals

### Task: Delete a Goal (Unwanted)
**Location:** Savings page
**Steps:**
1. Find goal in grid
2. Click delete button (trash icon)
3. Confirm deletion
4. Goal removed from list

### Task: Review Monthly Progress
**Location:** Dashboard or Analytics
**Steps:**
1. **Dashboard:** See KPI cards at top
2. **Analytics:** Get detailed breakdown
3. Check if spending is within budget
4. Check if savings are on track
5. Plan adjustments if needed

## ðŸ’¡ Pro Tips

### Tip 1: Link Dashboard to Savings
Start your monthly routine on Dashboard:
1. View your budget and spending
2. See savings goals preview
3. Click "+ Add Savings" if you have spare budget
4. Jump to Savings page for detailed management

### Tip 2: Use Categories Wisely
Create consistent expense categories:
- **Food** - groceries, restaurants
- **Transport** - gas, uber, transit
- **Entertainment** - movies, games, hobbies
- **Shopping** - clothes, gifts, misc
- **Utilities** - electricity, water, internet
- **Health** - doctor, gym, pharmacy
- **Education** - courses, books, tuition
- **Other** - everything else

### Tip 3: Goal Setting Strategy
Create 3 types of goals:
- **Short-term** (1-3 months) - vacation, gifts
- **Medium-term** (3-12 months) - car fund, laptop
- **Long-term** (1+ years) - house down payment

### Tip 4: Regular Check-ins
- **Daily:** Add expenses as you spend
- **Weekly:** Check Dashboard for trends
- **Bi-weekly:** Add savings on payday
- **Monthly:** Full review on Analytics page

### Tip 5: Budget Planning
- Set budget based on last month's average
- Leave 10-20% buffer for unexpected expenses
- Allocate remaining to savings goals
- Adjust monthly based on actual spending

## âš ï¸ Common Issues & Solutions

### Issue: Budget seems wrong?
**Solution:**
1. Go to Transactions page
2. Check if all expenses are recorded
3. Verify amounts are correct
4. Dashboard will recalculate automatically

### Issue: Goal progress not updating?
**Solution:**
1. Refresh page (Ctrl+F5 or Cmd+Shift+R)
2. Try the "+ Add Savings" button again
3. Check that amount is valid (> 0)
4. Success message should appear

### Issue: Can't find Dashboard?
**Solution:**
1. Click home icon in sidebar
2. Or click "Smart Budget" logo at top
3. Or navigate using back button

### Issue: Goal deleted accidentally?
**Solution:**
1. Refresh page (might restore if error occurred)
2. Otherwise, create goal again
3. Re-add savings amount
4. Keep better backups (contact support)
